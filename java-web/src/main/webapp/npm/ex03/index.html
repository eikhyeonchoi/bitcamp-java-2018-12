외부 라이브러리 사용 - 자바스크립트 패키지 관리자(npm)를 이용하여 라이브러리를 자동으로 다운로드 받기

    - nodejs의 npm 프로그램을 사용하면 CSS나 javascript 라이브러리를 자동으로 관리할 수 있다
      (gradle를 이용하여 자바 라이브러리를 자동으로 관리하듯)
    - 사용법:
      - 콘솔창에서 라이브러리를 사용할 웹 어플리케이션 루트(webapp) 디렉토리로 간다
      - '$ npm init' ==> 명령어를 실행한 디렉토리에 package.json, package-lock.json 파일 생성
      - '$ npm install bootstrap --save' 
        - 명령어를 실행한 디렉토리에 라이브러리를 다운로드 받을 node_modules라는 폴더가 생성된다
        - 지정된 라이브러리를 다운로드 받는다
        - 라이브러리 정보는 package.json 파일에 기록된다 
      - '$ npm install jquery --save'
        - bootstrap이 사용할 의존 라이브러리를 다운로드한다
      - '$ npm install popper --save'
      
      - '$ npm install' ==> package.json 에 등록된 라이브러리 자동으로 다운로드함
    
    [장점]
    - 외부 CSS, javascript 라이브러리를 자동으로 다운로드 할 수 있다
